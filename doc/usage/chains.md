# Language chains
The following are provided as chainable getters to improve the readability of your assertions.
They do not provide testing capabilities.

- `->and`
- `->at`
- `->be`
- `->been`
- `->but`
- `->does`
- `->has`
- `->have`
- `->is`
- `->of`
- `->same`
- `->still`
- `->that`
- `->to`
- `->which`
- `->with`
